TPS Racing: High-Performance Automotive Parts Store

TPS Racing is an automotive performance parts supplier. I led backend Magento 2 development including custom module creation, advanced product filtering, and secure third-party API integrations for inventory and fitment compatibility.

Key Achievements:

  • Developed a custom fitment module to help users find parts compatible with their vehicle models (Year, Make, Model filters).
  • Integrated a third-party API for real-time stock and product data synchronization.
  • Implemented layered navigation with performance-oriented filtering to improve UX for gearheads.
  • Created a custom import/export script using Magento models and repositories for massive SKU data updates.
  • Improved Magento performance and optimized database queries for large catalogs.
