FrontPoint: Enhancing eCommerce Efficiency through Custom Magento 2 Solutions

FrontPoint is a leading eCommerce platform offering high-quality products to customers. I was responsible for developing backend solutions to streamline quote management, optimize product pricing, and integrate external APIs for real-time updates.

Key Contributions:

  • Developed a custom Quote Management System to allow customers to create, manage, and submit quotes seamlessly.
  • Implemented a real-time inventory sync using third-party APIs, ensuring accurate product availability and pricing.
  • Optimized Magento 2 backend for better performance, reducing page load time and increasing scalability.
  • Designed and implemented custom frontend elements using KnockoutJS and AJAX for a smooth user experience.
  • Created and maintained custom APIs for product data synchronization, ensuring real-time accuracy.
